## How to Use

### Basic Setup
1. Open `index.html` in a web browser
2. The website is ready to use!

### Customization

#### Update Gallery Images
Edit the `galleryData` array in `script.js`:
```javascript
const galleryData = [
    {
        id: 1,
        title: 'Your Work Title',
        category: 'dresses', // or 'suits', 'casual'
        image: 'path/to/your/image.jpg'
    },
    // Add more items...
];
```

#### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2c2c2c;
    --secondary-color: #c9a96e;
    /* Customize other colors */
}
```

#### Update Content
- Edit text directly in `index.html`
- Update contact information
- Modify service descriptions
- Change company details

#### Add More Images
Replace the placeholder Unsplash URLs with your own images:
- Upload images to your server or use an image hosting service
- Update the image paths in both HTML and JavaScript

### Adding New Work to Gallery

To post new work:
1. Take high-quality photos of your work
2. Optimize images (recommended: 800-1200px width, WebP or JPEG format)
3. Add new entries to the `galleryData` array in `script.js`
4. Use appropriate categories: 'dresses', 'suits', or 'casual'

Example:
```javascript
{
    id: 13,
    title: 'Custom Wedding Dress',
    category: 'dresses',
    image: 'images/wedding-dress-01.jpg'
}
```
